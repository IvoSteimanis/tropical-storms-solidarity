# Replication Package: Moderate tropical cyclone exposure erodes solidarity needed for recovery

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21099060.svg)](https://doi.org/10.5281/zenodo.21099060)

## Overview

This package contains the code and data required to replicate the findings of "Moderate tropical cyclone exposure erodes solidarity needed for recovery" by Ivo Steimanis, Maximilian Burger, Bernd Hayo, Andreas Landmann, and Bjorn Vollan.

The analysis combines a rare panel dataset (2012, 2016, 2022) with a lab-in-the-field experiment to investigate the long-term effects of Super Typhoon Haiyan on solidarity and prosocial behavior in the Philippines. The global tropical-cyclone exposure analysis uses the TCE-DAT dataset (Geiger et al. 2018) with HYDE 3.2 population data.

## Quick start

The pipeline has two stages and **the Stata stage must run first**: `02_analysis_main.do` writes the margin and curve CSVs (into `results/intermediate/`) that the R scripts read to render Figures 2, 3 and 5.

- **Automated (Windows):** edit the two program paths at the top of `run_replication.bat`, then double-click it. It runs Stata (`run.do`) and then R (`run_R.R`) in the correct order.
- **Manual (any OS):** in Stata, `do run.do`; then in R, `source("run_R.R")`. (See full instructions below.)

## Data Availability Statement

The anonymized panel dataset and experimental data used in this study are provided in the `data/` directory. The data were collected by the authors under ethical approval from the Ethics Commission of the School of Business, Economics and Society at Friedrich-Alexander University Erlangen-Nurnberg. Geographic storm track data are publicly available from NOAA. Municipality-level damage statistics are from the Philippine National Disaster Risk Reduction and Management Council (NDRRMC). The TCE-DAT global tropical-cyclone exposure dataset (Geiger et al. 2018, Earth System Science Data, https://doi.org/10.5194/essd-10-185-2018) is publicly available from GFZ Potsdam (data DOI https://doi.org/10.5880/PIK.2017.008); because of its size (about 0.6 GB) it is not redistributed in this package and must be downloaded and extracted into `data/tce_dat/raw/` to regenerate the global exposure figure.

The data included here reproduce all statistical results, all tables, and Figures 2, 3 and 5 with no additional downloads. Reproducing the global tropical-cyclone exposure map (Figure 1) additionally requires the public TCE-DAT dataset (download from GFZ Potsdam, link above, and extract into `data/tce_dat/raw/`); the study-site typhoon-track map (Figure 4) uses the IBTrACS best-track file, which `scripts/R/12_study_site_figure.R` downloads automatically from NOAA NCEI if it is absent. Rendered versions of both maps are also provided in `results/figures/`.

## Directory Structure

- **`data/`**: Raw experimental, survey, and exposure data.
    - `PHI_Panel_12_16_22.dta`: Main panel dataset (2012, 2016, 2022 waves).
    - `distance_storm_km.dta`: Village-level distance to Typhoon Haiyan's eye.
    - `windspeed_predicted.dta`: Village-level calibrated sustained wind speed (knots), Holland (2008) wind-field model.
    - `windspeed_tcedat.dta`: TCE-DAT-based wind speed at village locations (robustness).
    - `!Lit_RQ2.xlsx`: Literature data for forest plot (Fig. S2).
    - `litfe_u_shape.dta`: Lab-in-the-field experiment data (transfer decisions by resource level and group size).
    - `map/`: Shapefiles for study area maps (`damages_municipalities.dta`, `muncoord.dta`, `StudySitePoints.dta`, `tycoord1.dta`).
    - `tce_dat/`: Global tropical-cyclone exposure data (TCE-DAT, Geiger et al. 2018).
        - `raw/`: TCE-DAT single-event CSV files (fix2015 variant, ~2,700 events globally 1950-2015) and IBTrACS v04r01 Western Pacific best-track data.
        - `processed/`: Intermediate R data objects (`.rds`), regenerated by `run_R.R`.
- **`processed/`**: Intermediate Stata dataset (`analysis_rdy.dta`) generated by `01_clean_data.do`. *Emptied and repopulated during a clean run.*
- **`scripts/`**: Analysis code.
    - **Stata do-files** (`01`-`03`): the full Stata pipeline.
        - `01_clean_data.do`: Data cleaning and variable construction -> `processed/analysis_rdy.dta`.
        - `02_analysis_main.do`: All main-text models and quoted numbers (transfers/reciprocity U-shapes, lab diffusion, ambiguity gap, per-10-kn validation), and the margin/curve CSV exports that R reads to render Figures 2, 3 and 5. Stata does **not** render the main-text figures.
        - `03_analysis_SI.do`: SI tables (S1-S33, incl. S19b/S19c) and SI figures (S2-S10, plus the ambiguity-gap robustness figure S11), and the TCE-DAT-vs-calibrated wind-speed equivalence diagnostics.
    - **R scripts** (`R/01`-`R/14`): global TCE-DAT exposure analysis plus the five main-text figures.
        - `R/01`-`R/10`: TCE-DAT extraction, grid-cell aggregation, headline numbers, and extended exposure analyses.
        - `R/11_combined_main_figure.R`: **Main text Figure 1** (global exposure, 3 panels).
        - `R/12_study_site_figure.R`: **Main text Figure 4** (study site map, 2 panels).
        - `R/13_figure2_3_maintext.R`: **Main text Figures 2 & 3** (field U-shape + mechanisms); re-plots the Stata exports from `02` (runs after the Stata pipeline).
        - `R/14_figure5_impacts.R`: **Main text Figure 5** (damage validation); re-plots the Stata lpoly curve exports from `02` (runs after the Stata pipeline).
    - `_toc_mediator_extension.do`: pre-registered exploratory mediation check; **not part of the pipeline**, retained for documentation only.
    - `clean_tex_tables.py`: post-processing of `esttab` `.tex` fragments (optional; not part of the analysis run).
    - `libraries/`: Bundled Stata packages (frozen adofiles).
- **`results/`**: Output tables and figures.
    - `tables/`: Regression tables (`.tex`, booktabs).
    - `figures/`: Main-text figures (R) and SI figures (Stata), `.png`.
    - `R_output/`: Additional R outputs (headline numbers, supporting CSVs, extended analysis figures).
    - `intermediate/`: Stata margin/curve CSVs (read by R) and `.gph` graph intermediates.
- **`run.do`**: Master Stata script (executes `01`-`03`). **Run this first.**
- **`run_R.R`**: Master R script (executes `R/01`-`R/14`).
- **`run_replication.bat`**: Windows driver that runs Stata then R in order.

## Software Requirements

### Stata

- **Stata 16 or later.** All scripts carry a `version 16` pin. A clean run regenerates every graph intermediate from scratch in-session, so no specific minor version is required. The analysis was last reproduced on StataNow 19 and on Stata 16/MP.
- **All required Stata packages are included** in `scripts/libraries/stata/`. `run.do` strips the default adopath and loads only these bundled versions, so no internet connection or manual installation is needed.
- **Memory**: the dataset is ~31 MB; any edition of Stata with at least 4 GB available memory suffices.

### R

- **R 4.3+** for the global exposure analysis and the five main-text figures.
- **Required packages**: `data.table`, `dplyr`, `sf`, `ggplot2`, `rnaturalearth`, `patchwork`, `haven`, `scales`, `showtext`, `ggspatial`.
- Install missing packages with: `install.packages(c("data.table", "dplyr", "sf", "ggplot2", "rnaturalearth", "patchwork", "haven", "scales", "showtext", "ggspatial"))`.

### General

- **Operating system**: results were last reproduced on Windows 11.
- **Estimated runtime**: Stata pipeline ~2-5 minutes; R pipeline ~5 minutes (standard desktop, 8 GB RAM).

### Bundled Packages (Stata)

To keep the package reproducible long-term, frozen versions of all user-written commands are included in `scripts/libraries/stata/`. Key packages: `utest` (Lind-Mehlum U-shape test), `iebaltab` (balance tables), `boottest` (wild cluster bootstrap), `sensemakr` (sensitivity analysis), `leebounds` (Lee attrition bounds), `estout` (table export), `grstyle`/`palettes`/`colrspace` (graph styling), `spmap`/`shp2dta`/`mif2dta`/`geo2xy` (maps).

## Instructions for Replication

### Setup

Extract the full directory structure to a local folder. Ensure `data/`, `scripts/`, `run.do`, and `run_R.R` are all at the same level.

### Step 1 (required first): Stata pipeline

In Stata, set the working directory to the package root and run the master script:

```stata
cd "C:/path/to/replication_package"   // adjust to your local path
do run.do
```

This runs `01_clean_data.do` -> `02_analysis_main.do` -> `03_analysis_SI.do` and produces:
- All quoted main-text numbers (logged to `scripts/logs/`).
- SI tables S1-S33 (`results/tables/*.tex`).
- SI figures S2-S11 (`results/figures/`).
- The margin/curve CSVs that R needs for Figures 2, 3 and 5 (`results/intermediate/fig2_*.csv`, `fig3_*.csv`, `fig5_*.csv`).

### Step 2: R pipeline

In R, set the working directory to the package root and run the master script:

```r
setwd("C:/path/to/replication_package")  # adjust to your local path
source("run_R.R")
```

This produces all five main-text figures:
- Figure 1 `results/figures/figure1_global_combined.png` (independent of Stata)
- Figure 4 `results/figures/figure4_study_site.png` (independent of Stata)
- Figure 2 `results/figures/figure2_field_ushape_windspeed_R.png` (from Step 1 CSVs)
- Figure 3 `results/figures/figure3_mechanisms_R.png` (from Step 1 CSVs)
- Figure 5 `results/figures/figure5_impacts_R.png` (from Step 1 CSVs)

plus headline numbers (`results/R_output/headline_numbers.txt`) and extended analysis outputs.

If `run_R.R` is run before `run.do`, Figures 2, 3 and 5 are skipped with an explicit message (the Stata CSVs they read are absent); just run `run.do` and re-run `run_R.R`.

### One-click alternative (Windows)

Edit the `STATA` and `RSCRIPT` paths at the top of `run_replication.bat`, then double-click it. It runs Steps 1 and 2 in order. In Windows batch mode Stata may take a few minutes to close its window after finishing; this is expected and the script waits for it.

### Verification

After both pipelines, `results/` should contain:

**Figures** (`results/figures/`, 17 `.png`; main-text and heterogeneity figures also exported as `.pdf`):
- **Main text, R-rendered (5):** `figure1_global_combined.png`, `figure2_field_ushape_windspeed_R.png`, `figure3_mechanisms_R.png`, `figure4_study_site.png`, `figure5_impacts_R.png`.
- **SI, Stata-rendered (11):** `figureS2_forestplot_literature.png`, `figS3`-`figS10` (differential attrition, PCA distributions, baseline trends, delta outcomes, long-term 2022, lab diffusion, perceptions, sensitivity), `figS12_raw_village_fits.png`, and `fig_ambiguity_gap_robustness.png` (Fig. S11).
- **SI, R-rendered (1):** `figureS_het_vulnerability.png` (pre-treatment heterogeneity by economic vulnerability).
- Fig. S1 (mechanisms diagram) is created manually and not generated by scripts.

**Tables** (`results/tables/`):
- 37 `.tex` table fragments (booktabs), covering the SI tables (incl. S19b/S19c, the continuous economic-vulnerability interaction table, and the coping-PCA component/loadings tables). Two descriptive tables are prepared separately and not generated by the scripts: the survey-studies overview (`tables/table_survey_studies.tex`, a manually compiled literature summary; source data in `data/!Lit_RQ2.xlsx`, sheet "Surveys") and the out-migration table (compiled from official documents).

**R outputs** (`results/R_output/`):
- `headline_numbers.txt` and supporting CSVs (`headline_by_regime.csv`, `headline_by_country_regime.csv`, `headline_mod_by_income.csv`, `relative_multiexposure.csv`), plus extended analysis figures.

## Data Description

### Main dataset: `PHI_Panel_12_16_22.dta`

A panel dataset from 30 rural coastal villages on Panay Island, Philippines, covering three waves:

| Wave | Year | Timing | N (individuals) |
|------|------|--------|-----------------|
| Baseline | 2012 | 1 year pre-Haiyan | 795 |
| Follow-up 1 | 2016 | 3 years post-Haiyan | 810 |
| Follow-up 2 | 2022 | 9 years post-Haiyan | 969 |

Balanced panel (waves 1-2): N = 450. Three-wave panel: N = 330.

### Key variables

| Variable | Description |
|----------|-------------|
| `mean_transfer` | Solidarity transfer amount (0-70 PHP), averaged across decisions in incentivized game |
| `windspeed_predicted` | Calibrated sustained wind speed (knots) at the village from Haiyan (Holland 2008 wind-field reparameterization of distance); primary exposure measure |
| `distance_storm_km_reversed` | Reversed distance (km) from village to Haiyan's eye; higher = closer (legacy exposure measure, retained for robustness) |
| `distance_cat` | Categorical exposure: 1 = low (<40 km), 2 = medium (40-80 km), 3 = high (>80 km) |
| `returner` | Indicator for participants present in both 2012 and 2016 |
| `vuln_cluster` | Vulnerability group: 1 = low, 2 = moderate, 3 = high (based on pre-disaster characteristics) |
| `z_recip` | Standardized reciprocity score (2022 wave) |

### Auxiliary datasets

| File | Description | Used in |
|------|-------------|---------|
| `distance_storm_km.dta` | Village-level distance to storm eye | `01_clean_data.do`, `R/12_study_site_figure.R` |
| `windspeed_predicted.dta` | Village-level calibrated sustained wind speed (knots), Holland (2008) | `01`-`03` (Stata); `R/12_study_site_figure.R` |
| `windspeed_tcedat.dta` | TCE-DAT-based wind speed at village locations (robustness) | `03_analysis_SI.do` |
| `!Lit_RQ2.xlsx` | Prior-literature data: `03` reads it for the Fig. S2 forest plot (scripted); the same workbook (sheet "Surveys") is the source for the manually compiled survey-studies overview table | `03_analysis_SI.do` (forest plot); manual (survey table) |
| `litfe_u_shape.dta` | Lab-in-the-field experiment data | `02_analysis_main.do`, `03_analysis_SI.do` |
| `map/*.dta` | Municipality boundaries, study site points, typhoon track | `R/12_study_site_figure.R` |
| `tce_dat/raw/` | TCE-DAT single-event CSV files (Geiger et al. 2018), fix2015 variant; IBTrACS v04r01 | `R/01`-`R/12` |
