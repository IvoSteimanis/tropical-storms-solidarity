static_figures/
================

Hand-made manuscript figures that are NOT produced by the analysis scripts
(run.do / run_R.R). They are shipped with the replication package and are NOT
deleted by a clean run (a clean run deletes /processed and /results only; see
run.do header). The Supporting Information references these figures directly
from this folder.

Contents:
  figureS1_ToC.png  -  Theory-of-Change diagram (Figure S1). Made manually in
                       PowerPoint. Referenced by submission SI_NCC.qmd.

If you add another hand-made figure to the manuscript, place it here (not in
/results/figures) so a clean run cannot lose it.
